# Utility Web Applications

A collection of utililies that may prove useful.

## POS-Generator

A small web-app for hand-generating POS-tags. Written in Scala with the (late, lamented) Scala-JS library.

### Running

1. Navigate to `Dramaturg/webapps/pos-generator/`
1. `python -m http.server 8080`
1. Visit `localhost:8080` in a browser.
